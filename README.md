# harkins-rss-feeds

This repo contains RSS feeds based off the movies currently playing in theaters. Feeds are updated once a week.

In order to use the feed in your RSS reader, copy the URL for the "raw" version of the file and add it to your reader. For instance, if you wanted to subscribe to "arizona-mills-w-imax", the url would be:
```
https://raw.githubusercontent.com/PatOConnor43/harkins-rss-feeds/refs/heads/master/rss/arizona-mills-w-imax.xml
```

For any other theater, simply replace "arizona-mills-w-imax" with the name of the theater you want to subscribe to. The list of theaters can be found in the rss folder.
